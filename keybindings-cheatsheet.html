WezTerm Keybindings

Personal config cheatsheet  ·  SUPER = Alt  ·  SUPER_REV = Alt+Ctrl  ·  LEADER = Alt+Ctrl+Space

SUPER = Alt
SUPER_REV = Alt + Ctrl
LEADER = Alt + Ctrl + Space (then press key within 1s)
Ctrl+Click = открыть ссылку
Разное / Интерфейс
Режим копирования F1
Палитра команд F2
Лаунчер F3
Лаунчер → Вкладки (Fuzzy) F4
Лаунчер → Воркспейсы (Fuzzy) F5
Полноэкранный режим F11
Debug Overlay F12
Поиск (нечувствит. к регистру) Alt+F
Открыть URL под курсором (QuickSelect) Alt+Ctrl+U
Курсор
Переход в начало строки Alt+←
Переход в конец строки Alt+→
Удалить всё до начала строки Alt+Backspace
Копирование / Вставка
Копировать в буфер обмена Ctrl+Shift+C
Вставить из буфера обмена Ctrl+Shift+V
Открыть ссылку под мышью Ctrl+Click
Вкладки
Новая вкладка (Default Domain) Ctrl+Shift+1
Новая вкладка (WSL) Ctrl+Shift+2
Дублировать вкладку (тот же домен + cwd) Ctrl+Shift+D
Закрыть вкладку Ctrl+Shift+W
Следующая вкладка Alt+]
Предыдущая вкладка Alt+[
Переместить вкладку вправо Alt+Ctrl+]
Переместить вкладку влево Alt+Ctrl+[
Переименовать вкладку (вручную) Alt+0
Сбросить имя вкладки Alt+Ctrl+0
Скрыть / показать Tab Bar Alt+9
Окно
Новое окно Alt+N
Развернуть окно (maximize) Alt+Ctrl+Enter
Уменьшить окно (-50px) Alt+-
Увеличить окно (+50px) Alt+=
Фон (Backdrop)
Случайный фон Alt+/
Выбрать фон (Fuzzy-список) Alt+Ctrl+/
Предыдущий фон Alt+,
Следующий фон Alt+.
Переключить фокус фона Alt+B
Панели (Panes)
Разделить панель вертикально Alt+\
Разделить панель горизонтально Alt+Ctrl+\
Zoom панели (развернуть/свернуть) Alt+Enter
Закрыть панель Alt+W
Перейти в панель ↑ Alt+Ctrl+K
Перейти в панель ↓ Alt+Ctrl+J
Перейти в панель ← Alt+Ctrl+H
Перейти в панель → Alt+Ctrl+L
Выбрать и поменять панель местами Alt+Ctrl+P
Прокрутка
Прокрутить вверх (5 строк) Alt+U
Прокрутить вниз (5 строк) Alt+D
Прокрутить вверх (¾ страницы) Shift+PageUp
Прокрутить вниз (¾ страницы) Shift+PageDown

Увеличить шрифт K
Уменьшить шрифт J
Сбросить размер шрифта R
Выйти из режима Esc / Q

Увеличить панель ↑ K
Уменьшить панель ↓ J
Уменьшить панель ← H
Увеличить панель → L
Выйти из режима Esc / Q
